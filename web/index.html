关于本站

本站简介

  • 本站叫：青鹿的小窝
  • 访问链接是：thecyandeer.com
  • 本站是花下青鹿的个人网站，非商业用途，主要是当个人名片和赛博花园使用。

主流的平台因为不同的社区调性，因此我发布的内容在不同平台会有所不同，这不利于我集中查阅自己发布的内容。因此，我想着搭建一个专属于自己的网站，就相当于在互联网上有了自己的独立空间、一块美丽的数字花园。在这里，我可以写写东西，记录生活，分享知识。你不觉得这很酷吗？所以，你可以在这里看到我所有发布在不同社交平台的内容：包括长视频作品、短视频作品、设定集文章、日常生活动态、二创作品转载、自己发布的公开资源等……反正就是我所有发布在社交平台的东西啦~
当然，本站主要是做关于我的内容整理，有点像档案馆的功能。而且本站内容不定期更新，相比于主流平台更新较慢。因此本站不具备任何商业用途和互动属性，所以看我的内容还是去主流平台上看吧~

技术栈

本站基于 Hexo 博客框架（MIT开源协议）和 Anzhiyu 主题（GPL-3.0开源协议）搭建，感谢大佬的开发让我可以专注于内容创作上~本站的代码在主题的基础上做了自定义更改，代码托管在 Github 上（除文章内容和影音图片外，遵循 GPL-3.0 协议）。

  • 用于网站的优化与文章分类，用户优化文章

  • 浏览数据的统计和数值展示

2、如何使用您的个人信息

本站为实现部分个性化功能，会在你的浏览器中进行本地存储，你可以随时清除浏览器中保存的所有 Cookies 以及 LocalStorage，不影响你的正常使用。

本站中的以下功能会在你的计算机上主动存储数据：
