# 青鹿小茶馆 - 花下青鹿个人网页

本人学识尚浅，此为自娱自乐，还请谅解！

本仓库仅为引入页的存放，具体可在“thecyandeer.com”上查看！

本个人网页由青鹿自主设计和开发，但……非常简陋，就只有非常简单的HTML、CSS和JS，没有使用任何复杂的框架或库。具体的工作流是：知识库软件编辑md文件，然后通过python读取md文件生成对应的html文件，最后就是自动整合和上传发布。网页是我自己提前设计的模板，非常简陋，但个人还是比较喜欢这种风格的hhh

这是因为我希望保持页面的轻量级和纯粹，绝对不是因为可以直接拿公选课期末大作业改hhh。

网站功能性不多，有个简单的首页（手机不太好适配）、感谢支持的页面以及“西岚屿档案馆”（用来给西岚屿世界观做wiki用）。为了使用方便，顺便开发了本地离线搜索功能、全适配日夜显示切换功能和小部分的手机适配。
